# Hangman Game
A beginner-friendly Python Hangman Game where players guess a randomly selected word one letter at a time. The project demonstrates Python fundamentals such as loops, conditional statements, strings, lists, the random module, and user input while providing an interactive console-based gaming experience.

## Problem Statement
Develop a text-based Hangman game where the player guesses a hidden word one letter at a time. The player is allowed a maximum of six incorrect guesses before the game ends.

## Key Features
- Randomly selects a word from a predefined list.
- Accepts one-letter guesses from the user.
- Displays correctly guessed letters and hides remaining letters.
- Limits incorrect guesses to six attempts.
- Prevents duplicate guesses.
- Displays win or game-over message.

  ## Concepts Used
  - Random
  - While Loop
  - If-Else
  - Strings
  - Lists
